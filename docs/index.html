A full image editing toolkit in a compact sidebar — zero dependencies, zero uploads.

✂

Crop

Interactive drag handles with 5 preset aspect ratios (Free, 1:1, 4:3, 16:9, 3:2) and manual pixel coordinate input for precision work.

↔

Resize

Scale by width, height, or percentage presets (25%–200%). Lock aspect ratio to prevent distortion. See dimensions side-by-side.

🎨

8 Filters + 8 Presets

Brightness, contrast, saturation, blur, grayscale, sepia, hue rotation, and opacity sliders. Plus one-click presets: Vivid, Warm, Cool, B&W, and more.

✨

12 Creative Effects

Unique pixel-manipulation filters like Memory Drift, Signal Corruption, Time Slice, and Genetic Mutation — with AI-powered recommendations.

💾

Export

Save as PNG, JPEG, or WebP with adjustable quality. One-click download with auto-generated filenames.

🌐

Page Image Detection

Scans the active tab and displays a thumbnail gallery of every image on the page. Click any image to edit it instantly.

Get started in 30 seconds

Install the extension

Get it from the Edge Add-ons store or load it unpacked from edge://extensions.

Open the side panel

Click the Image Editor icon in your toolbar. The editor opens in the browser sidebar.

Load an image

Browse files, drag-and-drop, paste from clipboard, or click any image detected on the current page.

Edit and export

Crop, resize, apply filters or creative effects, then download as PNG, JPEG, or WebP.
